# 📊 Price Tracker SaaS

A full-stack price tracking system that monitors product prices and sends alerts when prices drop.

## 🚀 Features
- Automated price scraping (Playwright)
- Price history tracking (SQLite)
- Alert system with SendGrid email
- Interactive dashboard with charts
- REST API using FastAPI

## 🛠 Tech Stack
- FastAPI
- SQLite
- Playwright
- SendGrid
- HTML/CSS/JavaScript
